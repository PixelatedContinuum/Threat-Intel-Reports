ZeroTrace Multi-Family MaaS Operation — Open Directory Exposure at 74.0.42.25
16 March 2026
Executive Summary
Analysis of an open directory at 74.0.42.25 revealed a fully operational Malware-as-a-Service platform hosting payloads from six distinct malware families. The infrastructure exposes active C2 configuration, staging scripts, and loader binaries indicating a sophisticated threat actor running concurrent campaigns against multiple target verticals.

Infrastructure Overview

The open directory at 74.0.42.25 was discovered during routine hunting operations. Port 80 served a browseable directory listing exposing 47 files across multiple subdirectories. The server fingerprint indicates nginx 1.18 running on Ubuntu, with directory listing enabled — an operational security failure that exposed the full toolkit.

File timestamps span a 72-hour window, suggesting active campaign operations. Several binaries were modified within hours of discovery, indicating the actor was actively staging payloads during the observation period.

Malware Families

Remcos RAT

Three Remcos samples identified across the directory. Configuration extraction revealed C2 at 185.220.x.x:4782 with keylogging and screen capture modules enabled. The samples are packed with a custom crypter not previously documented in public reporting.

XWorm v6

XWorm samples configured with Telegram-based C2 exfiltration. Bot token and chat ID hardcoded in the binary — both were active at time of analysis. The loader drops XWorm into a scheduled task for persistence under the name "WindowsDefenderUpdate".
